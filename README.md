Dynamic Profit  Dashboard

1.Project Title / Headline Insurence Analytics:
   A Profit Dashboard is  dynamic ,interactive data visulization tool to built to explore profit report data worldwide-focusing on Country wise Sales,Information about the Managers as well as profit wise country.

2.Short Description/Purpose:
    The Profit Dashboard is visually engaging and analytical Power bi report designed to help how many Managers are there and Country Wise sells with dynamic visuals this Dashboard shows the information about the profit of the company in the different visuals this is dynamic as well as interactive to the client.

3.Tech Stack The dashboard was built using the following tools and technologies:
* Power Bi Desktop-Main Data Visulisation Platform used for report creation.
* Power Query-Data Transformation and Cleaning Layer for reshaping and preparing the data.
* DAX(Data Analysis Expression)-Used for calculated Measures,Dynamic Visuals and Conditional Logic.
* Data Maodeling-Relationship established among tables.
